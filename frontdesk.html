Front-End Development

FrontDesk MS

I worked as the frontend developer on the FrontDesk Management System project, where I used React.js to design a dashboard that made it easy for users to track office information. The dashboard presented real-time data on office occupancy, meeting room scheduling, and visitor information, allowing users to make informed decisions about office usage. On the backend, the system was programmed using PHP and MySql, with an API used to retrieve and display the information on the dashboard.
In addition, the system featured a fully functional authentication system that allowed users to log in, recover their password, and log out. This was implemented using best practices in security to ensure that user data was kept safe and secure. Overall, my work on the FrontDesk Management System project was focused on creating a user-friendly and efficient system that helped users make the most of their office space and resources.
